# vimrc

## Install Plugins

```
curl -fsSL https://raw.githubusercontent.com/TheRealMattDunn/vimrc/main/plugins/dunnmj/colors/dunnmj-desert.vim > ~/.vim/pack/plugins/start/dunnmj/colors/dunnmj-desert.vim
curl -fsSL https://raw.githubusercontent.com/TheRealMattDunn/vimrc/main/plugins/dunnmj/plugin/dunnmj.vim > ~/.vim/pack/plugins/start/dunnmj/plugin/dunnmj.vim
```
